Using Terminal Multiplexers (Best for Development)
If you need to quickly run a script and check back on its output, use a multiplexer to keep the session "alive" even when you close your SSH connection. 
Tmux:
Start a session: tmux new -s mysession.
Run your app: python3 my_app.py.
Detach: Press Ctrl+B, then D.
Reattach later: tmux attach -t mysession.
Screen: Similar to Tmux; use screen to start, Ctrl+A then D to detach, and screen -r to reattach.
